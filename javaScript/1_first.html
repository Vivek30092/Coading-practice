basic of JS and variables in JS

IMP points

  • JS is dynamically typed language that is we don't need to define the type of variable before declaring it.

Variable naming rules

  • Variable names are case sensitive
  • only letter,digits, underscore,$ sign is allowed for declaring variable
  • first letter can't be number
  • Reserved words can't be variable names

naming convention in programming language

  1. fullName -> Camel Case
  2. fullname (not used)
  3. full_name -> snake case
  4. FullName -> pascal case
  5. full-name -> Kebab case

Variable declaration using [let, const & var]

var : Variable can be redeclared and updated. A global scope variable
let : variable can't be redeclared but can be updated. A block scope variable
const : variable can't be re-declared or updated. a block scope variable

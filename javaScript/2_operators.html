Operators in JS

    Arithmetic Operator
    1. +,-,*,/
    2. Modulus
    3. Exponentiation
    4. Increment
    5. Decrement
    Unary Operator
    1. Post Increment (Operator++)
    2. Post Decrement (Operator--)
    3. Pre Increment (++Operator)
    4. Pre Increment (--Operator)
    Assignment Operator
    1. a = a + 5, can be written as a+=5
    2. a = a - 5, can be written as a-=5
    3. a = a * 5, can be written as a*=5
    4. a = a / 5, can be written as a/=5
    comparison operators
    1. Equal to ==
    2. Not equal to !=
    3. equal to & type === (Strict Version means also check version)
    4. not equal to & type !==
    Note :
    double '==' doesn't check the datatype so we want datatype to be same then we use '==='
    Logical Operators : Answer only 'true' or 'false'
    1. Logical AND && ==> (to get true, both condition should be true)
    2. Logical OR || ==> (to get true, any one condition should be true)
    3. Logical NOT ! ==> (negation of condition)
    Ternary Operators (simpler, compact if-else)
      age > 18 ? "adult" : "not adult"
